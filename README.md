# FuckLib 自助图书馆助手

这是一个基于 FastAPI 和 React 开发的图书馆座位预约辅助系统，旨在简化图书馆座位的预约与签到流程。

## 主要功能

*   **座位预约**：支持查看座位布局并进行预约操作。
*   **自动任务**：支持设置定时任务（Cron），实现自动化预约和签到。
*   **状态监控**：实时查看座位状态和常用座位信息。
*   **消息推送**：集成 Bark 推送服务，及时接收预约成功或异常通知。
*   **用户管理**：支持多用户系统，包含邀请码注册机制。

## 技术栈

*   **后端**：Python, FastAPI, SQLAlchemy, APScheduler
*   **前端**：React, TypeScript, Vite, Tailwind CSS

## 快速开始

### 后端

1.  进入后端目录：`cd backend`
2.  安装依赖：`pip install -r requirements.txt`
3.  启动服务：`python main.py`

### 前端

1.  进入前端目录：`cd frontend`
2.  安装依赖：`npm install`
3.  启动开发服务器：`npm run dev`
